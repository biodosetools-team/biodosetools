# Load manual data

Load manual data

## Usage

``` r
load_manual_data(data_type, input, reactive_data, output, session)
```

## Arguments

- data_type:

  gamma or neutron

- input:

  what you upload in the UI

- reactive_data:

  access to the reactive data

- output:

  output for the UI

- session:

  let interaction with the shiny UI

## Value

manual curve information tables.
