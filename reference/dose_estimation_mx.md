# Prepare data to dose estimate using mixed fields manual

Prepare data to dose estimate using mixed fields manual

## Usage

``` r
dose_estimation_mx(input, reactive_data, data_type)
```

## Arguments

- input:

  what you upload in the UI

- reactive_data:

  access to the reactive data

- data_type:

  gamma or neutron

## Value

data for dose estimation using manual input.
