# Calculate yield

Calculate yield

## Usage

``` r
yield_fun(dose, general_fit_coeffs, protracted_g_value = 1)
```

## Arguments

- dose:

  Numeric value of dose.

- general_fit_coeffs:

  Generalised fit coefficients matrix.

- protracted_g_value:

  Protracted \\G(x)\\ value.

## Value

Numeric value of yield.
